# Snowboard 技巧

## 第一次上雪

- 要練習單腳 (前腳綁上) 對雪板的控制，習慣需要的力量和滑動的技巧
- 要能在緩坡先學會綁單腳 J turn，才能在下纜車的時候順利滑出去

## 後刃

因為當時是直接也沒請教練就上雪場，也忘記怎麼學會的了，不太清楚有什麼系統性的技巧，先跳過，等之後整理 CASI 的 [Quick-Ride System](https://members.casi-acms.com/documents/Ref_Guide.pdf) 再來補。

## 前刃

- 大原則

    - 有可能會有多種動作可以達成同樣的結果，但是要盡可能不費力地做出動作才是最好的動作
    - 用腳掌調整板子角度花費的肌肉只有腳掌的，會非常痠，但如果正確的使用前脛骨壓鞋舌的力量，就可以省力非常多。
    - 重心放在前刃邊緣上，眼睛不要看地上，要挺胸看向前方 (山上或前進方向的路上)，用手把屁股往前推推到極限，膝蓋向前跪來對雪板施力
    - 切忌踮腳尖，用腳掌控制角度
    - 向斜下方的啟動來自於把板子打斜，打斜的方向控制是透過腳掌的角度控制的，雪板的角度是靠膝蓋跪下去的幅度控制的，要往哪邊滑就讓哪邊的膝蓋往下跪多一點

- 強力落葉飄:
    
    - 必須在往下啟動時就開始減速，滑出來的形狀才會是開口朝山上的彎曲弧線
    - 減速控制都透過膝蓋加減壓，切勿使用腳踝改變角度來控制，會極度不穩定，最壞可能會不小心卡後刃往山下摔
    - 維持時時看向山下，也就是說到最低點的時候記得轉頭，同時獲得一個旋轉的力量，讓自己轉向
    - 要記得用折疊來改變加減速而非傾倒，傾倒較不穩定 (教練希望我們在這個初學階段不要有太多傾倒，似乎到更高級的時候做走刃會需要同時控制傾倒和折疊)。
    - 記得背挺直就可以改善上述問題。

- 紀錄:
    
    - 2025-10-16
    : 今天的自練稍微滑的更順了，有比較掌握到重心前中後的控制，在進行強力落葉飄時，要記得切換身體左右的重心，這點很常忘記，但是今天想到一個辦法，變成不是想著改變重心，而是以身體為主體稍微推動雪板，向右下啟動時重心要向板頭放，就想成是把腳往後施加推力，相對上身體重心就會跑到板頭了，希望隔天的上課也能順利過關。

    - 2025-10-17
    : 沒過。卡了很久才想起來，以前剛開始在練投手的時候也是，被說不太會察覺自己的動作，要更仔細觀察思考自己的動作細節並修正才會比較快練好，有點沮喪也很羨慕同時一起練的投手，比較早就可以在比賽中投球了。雖然後來好像也很少來練球了 (嗎?)

    - 2025-11-12
    : 終於通過了前刃這關，準備要學 C turn。

## 換刃

幾個重點:

- 整個過程維持
: 重心置於前腳 (可以想像自己要在前側膝蓋外側貼上貼紙，讓身體刻意向前傾斜多一點)，投轉注視前進方向，肩膀維持封閉 (平行雪板)
- 啟動
: 維持橫板狀態穩定，前腳擰板控制角度啟動下墜，直到板子轉到直板
- 重心移動
: 將重心放在前腳的**想去的邊刃上**，例如前轉後就要放在前腳腳跟上，不要放到板刃外面，容易摔，一旦刃咬住就會自動轉成功了。

- 紀錄
    
    - 2025-11-15
    : 去看 SB3 的課，得知通過目標是要完成穩定的 C turn，感覺不難，我也早就有 60% - 70% 的成功率了，打算自己自練過後拚一堂課通關。

    - 2025-11-19
    : 上完一堂 SB3 的課了，理解到換刃和 C turn 其實是兩件事，C turn 要能穩定做到 ***橫版打直 -> 換刃 -> 直板打橫***，大概就是 $C = PDP^{-1}$ 的關係，學習換刃的時候要認真感受維持直板咬到一邊刃時會有的切向，如果在雪機上會緩慢偏移，練習方式就是抓桿子在兩個刃之間練到能自由切換，如果在雪上為了減速避免暴衝就會慢慢打橫變成 J turn ($PD$)，就必須練習階梯式切雪道，使用同一個方向練習打直版 + 打橫板。如果在不熟的狀況做太快可能會忽略換刃的過程，導致雪板咬進雪裡就會摔。

    - 2025-11-23
    : 後轉前的時候容易卡刃，要確實轉成直板再踩想要去的那邊的刃，容錯蠻小的。***橫版打直*** 的啟動要**以腳踝主導**，控制單邊的腳踝角度讓那邊向下啟動，才能順利換到直板，而不是在這邊就把刃踩下去然後就會卡。

    - 2025-12-02
    : SB3 過關。還是要切記重心交給前腳，因為這樣前腳吃雪才容易導向，讓換刃後容易向正確方向切。視線看向前進方向，但是肩膀不要開，維持上半身大致平行雪板。換刃要轉移重心，但要讓重心踩在刃上，初學階段換刃不可超出雪板，否則容易摔。

## From LLM

### Phase 1（第一天上雪道：強制超慢大C turn）

目標：把雪機的耐心帶到雪道，先求穩不求連

專挑最寬、最緩的綠道（人少的最好）
每趟只做 6～10 個「超級大、超級慢的C turn」
每個轉彎都故意做到「板子完全平了、重心在中間、速度幾乎停」才換邊
動作口訣：雙手放胸前抱胸（防止甩上身）＋大聲數「1、2、3」才換邊
→ 這天你會滑得很「尷尬、很慢、很無聊」，但這就是在強迫身體把「做完一個轉彎」的感覺刻進肌肉記憶

### Phase 2（第二天：1.5個C turn + 縮短平板區）
目標：開始練習連結，但還是給自己保險

同樣緩綠道，但速度可以比昨天快20%
現在做「1.5個C turn」：第一個轉彎做完→故意等1秒平板區，再做下一個
重點口訣：「板子沒平，我就不換邊」
後腳完全放鬆，只用前腳壓邊（想像後腳是斷掉的）
一趟能把平板區縮到0.5秒內就算成功

### Phase 3（第三天起：真S turn + 數拍子進階）

目標：把平板區縮到看不見

可以移到中緩藍道
現在開始數「1、2」就換邊（比昨天更快）
視線永遠看「下一個轉彎要去的樹/人/標誌」，強迫上身分離
每當你覺得要急的時候，就故意再做大C turn 2～3個，把速度殺掉重來
一旦你發現「欸我連續10個轉彎都沒急」，恭喜你，已經突破中級牆

超實用三招「現場救急神技」（你一定會用到）

感覺要急或卡住 → 立刻雙手抱頭＋大C turn殺速度，重新開始
後腳開始亂扭 → 故意把後腳抬起來離板1秒（真的抬！），強迫前腳獨立工作
上身開始甩 → 把雙手放到屁股後面交叉（像被銬住），只能用下半身轉

最後提醒

第一天你一定會覺得「我滑得好爛好慢」，但這是正常的！第二天你就會突然覺得「怎麼突然順了很多」，第三天就會真正開竅。
不要跟旁邊三四天就S turn的人比，他們99%都是假動作，你現在練的是十年後還能用的真功夫。


「**單腳滑行練習（One-foot skiing / J-turn drill）**」，專門用來治「甩上身、後腳亂扭、前腳不敢壓」這三個中級牆元兇。  
很多日本、加拿大、歐洲的教練系統，都把這個當成「從C turn升S turn的必經關卡」。

### 你現在最需要的三種「只綁前腳」練習（照順序練）

1. **最基礎：直滑 + 煞停（練前腳獨立壓邊）**  
   - 只綁前腳（後腳完全拔出來踩在板子上或空著）  
   - 從最緩的綠道頂端開始直滑  
   - 用前腳腳趾或腳跟壓邊，讓板子自己轉90度到完全停  
   - Toeside 和 Heelside 各練10次  
   → 這招會讓你立刻知道「原來前腳真的可以獨立控制整塊板」

2. **進階：單腳大C turn（J-turn）**  
   - 還是不綁後腳  
   - 從直滑開始，用前腳慢慢壓邊，畫出一個超級大的「J」字（從直線變成橫向停住）  
   - 要求：不能用任何上身幫忙，雙手抱頭或抱胸  
   - 兩邊各練10～15次，直到你可以用前腳畫出漂亮的半圓  
   → 這就是真正「前腳主導」的感覺，之後綁雙腳做S turn時會自動帶出來

3. **終極：單腳連續S turn（超難但超有效）**  
   - 後腳踩在板子上但完全不綁（或直接拿掉）  
   - 試著用前腳連續做3～5個S turn  
   - 一開始一定歪歪扭扭，但只要能連三個就算過關  
   → 很多人練完這招後，綁回後腳時會突然覺得「怎麼突然變超順？」

### 實戰建議（明天直接可以用）
- 第一趟先找最寬最緩的綠道，旁邊沒什麼人  
- 把後腳從綁帶拔出來，直接做「單腳J-turn」來回10趟  
- 練完再把後腳綁回去，你會立刻感覺到：
  - 前腳變超主動
  - 後腳不敢亂扭了
  - 上身完全不敢甩（因為一甩就摔）

### 每天必備的「三寶檢查清單」（滑完一趟就自問）
1. 上半身有沒有比板子先轉？→ 沒有＝過
2. 後腳有沒有偷扭尾刀？→ 沒有＝過
3. 每個轉彎有沒有真的做完？→ 有＝過

三個都過，這趟才算數；只要有一個沒過，就再滑回起點重來。

### 裝備小調整（很重要）
- 把後腳綁帶調鬆一格（讓後腳真的不敢用力）
- 站姿角度可以試著把後腳角度調小一點（例如 +15 / 0 或 +12 / -3），比較容易前腳主導
- 膝蓋永遠保持微彎，屁股不要坐後座
