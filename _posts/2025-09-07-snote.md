# 單板滑雪（Snowboard）技巧筆記

## 第一次上雪

- 先練習**單腳滑行**（前腳綁定，後腳推雪推進）
  - 目的是熟悉雪板控制所需的力量、平衡與滑動感覺
- 必須先在緩坡練會**單腳 J turn**（前刃或後刃），才能順利從纜車下場區滑出，避免卡住或失控

## 後刃（Heelside）技巧

因為一開始沒請教練、直接上雪，當時的學習過程較混亂，尚未系統化整理。  
建議之後參考 CASI 的 **QuickRide System**（5階段：Basics → Sliding → Control → Turning → Flow）再回頭補充後刃基礎部分。目前先跳過，留待未來更新。

## 前刃（Toeside）技巧

### 大原則

- 動作選擇以「**最省力、最有效率**」為優先（相同結果下，選肌肉負擔最小的做法）
- **避免只用腳掌踮腳尖**控制角度（容易痠、控制不精準、不穩定）
  - 正確方式：用**前脛骨前壓鞋舌**（shins press tongue），搭配膝蓋前跪，省力非常多
- **重心**放在前刃邊緣上
  - 眼睛**挺胸看向前方**（山上或前進方向），**不要低頭看板或地面**
  - 用手把**臀部往前推到極限**，膝蓋向前跪，對雪板施加壓力
- 啟動與方向控制
  - 斜下啟動來自**板子打斜**（角度由腳踝/腳掌控制）
  - 彎度大小由**膝蓋跪下幅度**決定
  - 想往哪邊滑 → 那邊膝蓋跪多一點（不對稱施壓）

### 強力落葉飄（Power Pendulum / Aggressive Falling Leaf）

- 關鍵：在**往下啟動瞬間就開始減速**，才能滑出**開口朝山上**的圓弧形軌跡（像微笑曲線）
- 減速完全靠**膝蓋加壓 / 釋壓**控制，**絕對不要用腳踝改變角度**（極不穩定，容易突然卡後刃向山下摔）
- 視線維持**看山下**：到最低點時**轉頭**，同時產生自然旋轉力幫助換方向
- 用**折疊（flexion/extension）**改變加減壓，而非**傾倒（inclination）**（初學階段傾倒不穩定，教練通常建議先避免；更高階走刃時才會結合傾倒與折疊）
- **背部挺直**能大幅改善姿勢穩定性

### 個人紀錄

- **2025-10-16**  
  自練進步較明顯，開始掌握重心前中後移動。強力落葉飄時常忘記切換左右重心。今天想到一個訣竅：不是刻意「移重心」，而是以身體為主體推動雪板。例如向右下啟動時，把腳往後推 → 相對重心就自然前移到板頭。希望明天上課能順利過關。

- **2025-10-17**  
  沒過關。卡很久才想起，以前練投手時也被說「不太能察覺自己動作」，需要更細緻觀察與修正。過程有點沮喪，也羨慕同期練投的同伴比較快能上場比賽（雖然後來他們好像也沒常練了？）

- **2025-11-12**  
  終於通過前刃關卡，準備進入 C turn 階段。

## 換刃（Edge Change）

### 幾個重點

- 整個過程維持：
  - 重心放在**前腳**（想像前膝外側貼一張貼紙，讓身體刻意前傾）
  - 視線投射**前進方向**
  - 肩膀**封閉**（平行雪板，不要開肩）
- 啟動：
  - 先維持橫板穩定 → 用**前腳腳踝主導擰板**控制角度，讓板子往下墜，直到轉成**直板**（fall line）
- 重心移動：
  - 將重心移到**前腳的目標邊刃**上（例如 Regular 站姿，從前刃轉後刃 → 重心放到左腳腳跟）
  - **不要放到板外**（容易摔）；一旦刃正確咬住雪，板子會自動帶動轉向
- 轉完後穩定：
  - 使用「**上帝與老人技巧**」

### 個人紀錄

- **2025-11-15**  
  旁聽 SB3 課程，目標是完成穩定 C turn。感覺不難，自己已有 60–70% 成功率，打算自練後直接挑戰一堂課通關。

- **2025-11-19**  
  上完 SB3 課後才搞懂：**換刃** 和 **C turn** 是兩件事。  
  C turn 要穩定做到「**橫板 → 直板 → 換刃 → 直板 → 橫板**」（類似 $C = PDP^{-1}$ 的循環）。  
  練習換刃時要認真感受「直板咬單邊刃」的切向力（雪機上會緩慢偏移）。  
  雪上為了減速常不自覺打橫變 J turn，建議用**階梯式切雪道**、同方向反覆練「打直 + 打橫」。  
  如果太急，容易忽略換刃過程 → 刃突然咬進雪裡直接摔。

- **2025-11-23**  
  後轉前容易卡刃。關鍵：一定要先確實轉成**直板**，再踩目標邊刃（容錯很小）。  
  「橫板打直」的啟動要**以腳踝主導**，單邊腳踝角度控制向下啟動，而不是一開始就踩刃（會卡）。

- **2025-12-02**  
  SB3 過關。  
  切記重心交給**前腳**，前腳吃雪才容易導向，讓換刃後順利切正確方向。  
  視線看前進方向，肩膀不要開，上半身大致平行雪板。  
  換刃時重心要踩在刃上，**初學階段不可超出雪板**，否則易摔。

- **2026-01-01 ~ 2026-01-06**  
  連續一週雪上練習，已能熟練綠線 **S turn**，藍線可完成單個 C turn，紅線容易失控。仍需慢慢推進。

## Carving（刻滑 / 刃滑）

- 相較 skid turn（搓雪轉彎），Carving 更講究**重心置中**，左肩（或對應邊）略低，肩膀與斜坡平行
- 換刃時重心可移到**另一邊刃外側**（skid turn 初學則要求不超出刃，否則無足夠向心力）
- 後刃：維持**腳尖抬起**感覺，提高立刃角度，前腳膝蓋不用彎太多
- 前刃：膝蓋需彎更多來增加立刃角度
- 大原則：**降低重心**（身體縮小感），但**頭保持正直看前方**。若非開放式姿勢，下巴貼近前腳側肩膀，用視線引導，避免上身多餘轉動

### 練習方式

1. 直板加速 → 前刃急煞 → 水平向上跳，每跳都要用前刃壓出明顯痕跡
2. 直板加速 → 後刃急煞 → 水平向下跳，每跳都要用後刃壓出明顯痕跡
3. 斜下切坡，卡一下卡一下留下間斷刃痕（前刃後刃都要練）
4. 斜下切坡後用速度壓好刃，繼續往高處回山，練習減速狀態下壓出長距離連續刃痕
5. 練習大彎：每個水平處放平 → 重心換到另一刃，視線維持看行進方向（切勿看山上/山下，會改變意識導致失敗）
6. 每個彎最好轉完整半圓，換刃處速度方向平行等高線

### 個人紀錄

- **2026-01-30 ~ 2026-02-07**  
  藍紅線可搓雪小迴轉，亂雪或不整地容易失控。紅線以下能簡單用刃滑行，但暫無法精準控制彎形大小與速度。紅藍線 Carving 一兩個彎後速度過快必須煞車；綠線可多幾個彎但仍會害怕。  
  問題可能出在蹲不夠低，施壓與釋放不正確。綠線低速時可 hop ollie，黑線可一個一個 C turn 下，但亂雪仍害怕。
