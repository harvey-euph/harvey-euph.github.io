## 單字

### 冠詞 (artículo)

- artículo definido ( 相當於 the )

|      | 單數 | 複數  | 例子                               |
| ---- | ---- | --- | -------------------------------- |
| 陽性 | el | los | el libro / los libros (這本書 / 這些書)  |
| 陰性 | la | las | la casa / las casas (這間房子 / 這些房子)  |

- artículo indefinido ( 相當於 a, an, some )

|      | 單數 | 複數   | 例子                            |
| ---- | --- | ---- | ----------------------------- |
| 陽性 | un  | unos | un libro / unos libros (一/一些書)  |
| 陰性 | una | unas | una casa / unas casas (一/一些房子)  |

### 指示詞 (demostrativos)

|              | 陽性單數 | 陽性複數 | 中性 | 陰性單數 | 陰性複數 |

| **那 (中距)** | ese   | esos     | eso     | esa     | esas     |

### 頻率副詞

| **siempre**          | always          | *Siempre tomo café en la mañana.*                |
| **casi siempre**     | almost always   | 口語裡常用來「稍微弱化 siempre」：*Casi siempre estudio en casa.* |
| **a veces**          | sometimes       | 非常常見，日常隨口會說：*Voy al cine a veces.*                   |
| **de vez en cuando** | once in a while | 比 *a veces* 更輕：*De vez en cuando salgo a correr.*    |
| **casi nunca**       | hardly ever     | 很常見的補充型說法：*Casi nunca cocino.*                       |
| **nunca**            | never           | *Nunca fumo.*                                 |

### 介詞 (preposiciones)

#### a 方向 / 對象 / 時間點

- 方向 / 去到某地：Voy a la escuela. (我去學校) 
- 人作為動作對象 (人前面要用 a，叫「a 人稱」) ：Veo a María. (我看到瑪麗亞) 
- 時間點：La clase empieza a las ocho. (課在 8 點開始) 
- 方式 / 速度：Vamos a pie. (我們走路去) 

a veces (有時候) 
a menudo (經常) 
a pie (步行) 
a mano (在手邊、手工) 
a tiempo (準時) 
a la una / a las dos (在一點/兩點鐘) 
ir a + 動詞原形 (去做某事 → 未來式表達：Voy a estudiar.) 

- aprender a + inf. → 學會做某事
    - Aprendo a conducir. (我在學開車) 
- empezar a + inf. → 開始做某事
    - Empezamos a estudiar. (我們開始學習) 
- ir a + inf. → 將要做某事 (未來表達) 
    - Voy a viajar mañana. (我明天要旅行) 
- acostumbrarse a + 名詞/inf. → 習慣於…
    - Me acostumbré a vivir aquí. (我習慣住在這裡) 

#### de 來源 / 所屬 / 部分

- 來源 / 出處：Soy de México. (我來自墨西哥) 
- 所有格：El libro de Juan. (胡安的書) 
- 部分：Un vaso de agua. (一杯水) 
- 材料：Una mesa de madera. (木桌) 

de repente (突然) 
de nuevo (再一次) 
de vez en cuando (偶爾) 
de nada (不客氣) 
tener ganas de + 動詞 (想要做某事) 
tratar de + 動詞 (試著做某事) 
acabar de + 動詞 (剛剛做完某事) 
acabar de + inf. → 剛剛做完某事
Acabo de comer. (我剛吃完) 

- dejar de + inf. → 停止做某事
    - Dejé de fumar. (我戒菸了) 
- tratar de + inf. → 嘗試做某事
    - Trato de aprender francés. (我試著學法文) 
- acordarse de + 名詞/inf. → 想起…
    - Me acuerdo de ti. (我想起你了) 

#### en 位置 / 交通工具 / 時間範圍

- 地點 / 在...裡：Estoy en casa. (我在家) 
- 交通工具：Voy en tren. (我搭火車) 
- 月份 / 年份 / 時期：Nací en 1990. (我出生於 1990 年)
- en la calle (在街上) 
- en serio (認真地，真的嗎？) 
- en general (通常、一般而言) 
- en seguida (馬上)
- pensar en + 名詞/inf. → 想到 / 考慮
    - Pienso en mi familia. (我在想我的家人) 
- creer en + 名詞 → 相信…
    - Creo en Dios. (我相信上帝) 
- entrar en + 名詞 → 進入…
    - Entramos en la casa. (我們進入房子) 

#### con 一起 / 工具 / 特徵

- 和某人：Salgo con mis amigos. (我和朋友出去) 
- 工具 / 手段：Escribe con un bolígrafo. (用原子筆寫) 
- 方式 / 特徵：Una chica con ojos azules. (一個藍眼睛的女孩) 
- con mucho gusto (樂意之至)

- con cuidado (小心地)
- con razón (有道理) 
- estar con alguien (支持某人) 
- soñar con + 名詞/inf. → 夢見 / 渴望…
    - Sueño con viajar a Japón. (我夢想去日本旅行) 
- enojarse con + 人 → 對某人生氣
    - Me enojé con él. (我對他生氣) 
- casarse con + 人 → 與某人結婚
    - Se casó con Ana. (他跟安娜結婚了) 

#### por 原因 / 經過 / 交換 / 時段

👉 很多時候對應英文的 by / through / because of / per

原因：Lo hice por ti. (我為了你做的) 

途經 / 經過：Pasamos por el parque. (我們經過公園) 

交換 / 價格：Te doy 10 euros por el libro. (我用 10 歐元買這本書) 

時間長度 / 時段：Estudié por tres horas. (我學了三小時) 

被動語態：El libro fue escrito por Cervantes. (書是塞萬提斯寫的) 

por ejemplo (例如) 
por fin (終於) 
por supuesto (當然) 
por eso (因此) 
gracias por + 名詞/動詞 (感謝…) 

- preocuparse por + 名詞 → 為…擔心
- Me preocupo por mis hijos. (我為孩子們擔心) 
- interesarse por + 名詞 → 對…有興趣
    - Me intereso por la historia. (我對歷史有興趣) 
- preguntar por + 人/事 → 詢問關於…
    - Preguntaron por ti. (他們問起你了) 

#### para

核心意思：目的 / 方向 / 期限
👉 常對應英文的 for / in order to / to

目的：Estudio español para viajar. (我學西文為了旅行) 

用途：Un vaso para agua. (一個裝水的杯子) 

收件人：Este regalo es para ti. (這禮物是給你的) 

方向 / 去往：Salimos para Madrid. (我們出發去馬德里) 

期限 / 截止：La tarea es para mañana. (作業是明天的) 

para siempre (永遠) 

para nada (根本不，一點也不) 

listo/a para + 動詞 (準備好去做…) 

trabajar para + 名詞 (為…工作) 

estudiar para + 動詞 (學習為了…) 

salir para + 地點 (出發去某地) 

servir para + 名詞/inf. → 用於…

Este cuchillo sirve para cortar pan. (這刀是用來切麵包的) 

trabajar para + 名詞 → 為…工作

Trabajo para una empresa internacional. (我為一間國際公司工作) 

#### sin 沒有 / 缺少

沒有：Café sin azúcar. (無糖咖啡) 

缺少某物 / 人：No puedo vivir sin ti. (我不能沒有你) 
sin duda (毫無疑問) 

sin embargo (然而，不過 → 連接詞性質) 

estar sin + 名詞 (缺少某物) 

vivir sin + 名詞 (沒有...生活) 

irse sin + 動詞原形 (沒有做某事就離開：Se fue sin despedirse. 他沒道別就走了) 

salir sin + inf. → 沒做某事就離開

Salió sin despedirse. (他沒道別就走了) 

- estar sin + 名詞 → 缺少...
    - Estoy sin dinero. (我沒錢) 

#### sobre 

| 用法              | 中文意思     | 例句                                                               |
| ----------------- | ----------- | ---------------------------------------------------------------- |
| **位置 / 在…上面** | 在...上面    | El libro está **sobre** la mesa.（書在桌子上）                          |
| **關於 / 內容**   | 關於...      | Hablamos **sobre** política.（我們談論政治）                             |
| **數量 / 超過**   | 超過、比...多 | Hay más de 50 personas **sobre** 60 esperadas.（有超過 60 人中約 50 人到） |

常見片語 / 慣用法

información sobre … → 關於…的資訊

Necesito información sobre el curso.（我需要關於課程的資訊）

discutir sobre … → 討論…

Discutimos sobre la película.（我們討論電影）

un libro sobre … → 一本關於…的書

Compré un libro sobre historia.（我買了一本歷史書）

如果想表示 位置 → 在…上面 → 物理接觸或覆蓋 → sobre = on / on top of

如果想表示 主題 / 關於… → 抽象 → sobre = about / regarding

## 文法

### 形容詞等級變化

#### 比較級 (Comparativo)

- 優等比較 (más ... que = more ... than) 
    - Este coche es más rápido que aquel.
    : 這輛車比那輛更快。
    - Ella es más inteligente que su hermano.
    : 她比她弟弟更聰明。
    
- 劣等比較 (menos ... que = less ... than) 
    - Esta ciudad es menos grande que Madrid.
    : 這個城市比馬德里小。
    - El libro es menos interesante que la película.
    : 這本書比電影沒那麼有趣。

- 同等比較 (tan ... como = as ... as) 
    - Juan es tan alto como Pedro.
    : Juan 和 Pedro 一樣高。
    - Ellos tienen tantos libros como nosotros.
    : 他們有跟我們一樣多的書。
    - tan + 形容詞/副詞 + como (形容詞/副詞比較) 
    - tanto/a/os/as + 名詞 + como (名詞比較，要配合性數) 

#### 最高級 (Superlativo relativo)

表示「最...」，通常用: 定冠詞 + 名詞 + más/menos + 形容詞 + de ...

- María es la más alta de la clase.
: María 是班上最高的。
- Este hotel es el más caro de la ciudad.
: 這家飯店是城裡最貴的。
- Son los menos interesantes de todos.
: 他們是所有人中最不有趣的。

#### 超級 ( Superlativo absoluto )

表示「非常...」，相當於英文的 very, extremely。
做法是把形容詞結尾 加 -ísimo/-ísima/-ísimos/-ísimas。

例子：

- guapo → guapísimo (超帥) 
- inteligente → inteligentísimo (非常聰明) 
- grande → grandísimo (非常大) 
- fácil → facilísimo (非常簡單) 

特殊注意拼字變化：

- rico → riquísimo (去掉 -co，加 -quísimo)
- largo → larguísimo (-g- → -guísimo)
- feliz → felicísimo (-z → -císimo)

#### 特殊 / 不規則變化

| 原級              | 比較級   | 最高級      | 中文 |
| ---------------- | ----- | -------- | ------------------ |
| bueno (好)        | mejor | el mejor | 好 → 更好 → 最好        |
| malo (壞)         | peor  | el peor  | 壞 → 更壞 → 最壞        |
| grande (大, 年長)  | mayor | el mayor | 大 → 更大/年長 → 最大/最年長 |
| pequeño (小, 年幼) | menor | el menor | 小 → 更小/年幼 → 最小/最年幼 |


## 一些剛好看到覺得不錯的句子

Gracias por invitarme.  
: 謝謝你邀請我

Puedes ayudarme.  
: 可以幫我一下嗎

Hay cosas que no sé si son verdaderas o falsas, pero si me va a ayudar en algo, las adopto.  
: 有些事我也不知道是真是假，但只要對我有幫助，我就信啦

Ya llevo mucho tiempo sin enfermarme.  
: 我好久沒生病了耶~

No te preocupes, ya toqué madera.
: 別怕，我已經敲木頭了 (迷信用法)

Estar con alguien de otro país es increíble, pero también está lleno de desafíos que nadie te cuenta.  
: 跟來自另一個國家的人在一起很棒，但也充滿了許多沒人提過的挑戰

¿Tú eres idiota?  
: 你是白癡嗎?

Odio a las personas que tratan de imponer su autoridad en la vida de otra persona.
: 我超討厭那種喜歡把自己的意見硬塞到別人人生裡的人

Te Amé, Te Amo y Te Amaré
: 以前愛你 現在愛你 以後也愛你

¿Dónde has ido?
: 阿你剛剛去哪阿

Me quede en el carro durmiendo
: 我在車上睡覺

¿tú consideras que eres una fanatica de la música?
: 你覺得自己算是音樂狂熱份子嗎?

¿Consideras que tienes un gusto exquisito en la música?
: 你覺得你的音樂品味好嗎?

¡Qué interesante!
: 太有趣了!

¡Exactamente!
: 對嘛

Acabamos de comer, asi que creo que es el mejor momento para conversar.
: 我們剛吃完飯，所以我覺得現在是聊天的最好時機。

Bienvenida a Latinoamerica.
: 歡迎來到拉丁美洲

Ese coctel sabia muy fuerte a alcohol para mi.
: 那杯雞尾酒對我來說太烈了

Yo soy una persona muy extrovertida.
: 我是一個很外向的人

Espanol... Parece que muchas personas hablan eso.
: 感覺很多人會講西文

Y encima, yo era una persona perfeccionista.
: 更糟的是，我以前還是一個完美主義者。

Yo no tengo miedo de hablar frente a las personas, pero soy malo aprendiendo idiomas. 
: 我不怕在大家面前講話，但我沒有學習語言的天分

Aparte de tomar los cursos internacionales para aprender espanol, tambien pagaba una tutora para que hablara conmigo. 
: 除了上語校的課程學西語之外，我還請了一個家教來陪我聊天

El camino al infierno se siente como el cielo, y el camino al cielo se siente como el infierno.
: 通往地獄的路感覺像天堂，但通往天堂的路卻常常像地獄

Tu hermana tiene mas conocimiento que tu
: 你妹比你更有常識
